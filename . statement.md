# Project Statement  
## Password Generator Using Python

### Problem Statement  
Many users create weak passwords that are easy to guess and unsafe. Creating strong and complex passwords manually is difficult for them.  
This project provides a simple Python application that generates secure and random passwords using a graphical user interface.

---

### Scope of the Project  
The project focuses on developing a desktop-based password generator using Python and Tkinter.  
It allows users to generate strong random passwords and copy them easily.  
It does not include password storage or online features.

---

### Target Users  
This application is designed for:
- Students  
- Office professionals  
- Developers  
- General computer users  
- Non-technical users  

---

### High-Level Features  
- Random strong password generation  
- Password length selection  
- Generate single or multiple passwords  
- Copy to clipboard  
- Simple and user-friendly GUI  
- Works offline
